# План разработки — сайт автосалона Suzuki

**Срок:** 12 недель  
**Стек (рекомендованный):** Next.js + NestJS + MySQL  
**Последнее обновление:** 2026-06-19

## Структура документации

План разбит на **6 фаз** (папки `phase-01` … `phase-06`). Внутри каждой фазы — **эпики** (папки `E*`), внутри эпика — **задачи** (отдельные `.md` файлы).

## Фазы

| Фаза | Недели | Milestone | Папка |
|------|--------|-----------|-------|
| 0. Фундамент | 1–2 | Архитектура, дизайн, CI/CD | [phase-01-foundation](./phase-01-foundation/) |
| 1. Каталог | 3–4 | Главная + каталог + фильтры | [phase-02-catalog](./phase-02-catalog/) |
| 2. Конверсия | 5–6 | Конфигуратор + бронирование | [phase-03-conversion](./phase-03-conversion/) |

## Definition of Done (общий)

- [ ] Код в репозитории, PR прошёл review
- [ ] Адаптив: mobile / tablet / desktop
- [ ] Нет критичных ошибок в консоли и логах
- [ ] Базовые тесты для API и ключевой логики
- [ ] Alt-теги, aria-labels где необходимо
- [ ] Endpoint задокументирован в Swagger
- [ ] Проверка: Chrome, Firefox, Safari, Edge (2 последние версии)
