# Контент-заглушки и референс

## Референс

**Сайт:** [https://suzuki-motor.ru/](https://suzuki-motor.ru/)

Используем как ориентир для:

- структуры разделов и навигации;
- подачи модельного ряда;
- тона текстов (официальный, лаконичный);
- визуальной иерархии (hero, блоки моделей, CTA).

> Мы **не копируем** дизайн и контент один в один — только берём паттерны для MVP с минимальным наполнением.

---

## Минимальное наполнение (seed)

| Раздел | Заглушка |
|--------|----------|
| Главная | Hero + 3–4 модели + CTA «Тест-драйв» |
| Каталог | 5–8 моделей Suzuki (Vitara, Jimny, Swift и т.д.) — публичные характеристики |
| Б/у | 2–3 карточки-примера |
| Акции | 1–2 промо-блока |
| О дилере | **Name name** — Lorem + placeholder фото |
| Контакты | **Name name**, г. Москва (заглушка), +7 (000) 000-00-00, info@name-name.ru |
| FAQ | 5–10 типовых вопросов |
| Блог | 2–3 статьи-заглушки |
| Отзывы | 3 отзыва без реальных имён |
