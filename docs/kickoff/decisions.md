## 1. Платформа (п.5 ТЗ)

| Решение | Значение |
|---------|----------|
| **Выбранный вариант** | Кастомная разработка: **Next.js (React) + NestJS (Node.js)** |
| Соответствие ТЗ | ✅ Да — п.5 явно допускает «React + Node.js» |
| CMS для контента | Headless (Strapi или admin в NestJS) — не WordPress, не 1С-Битрикс как сайт |

> **Важно:** «Битрикс» в решениях ниже — это **CRM Битрикс24**, а не CMS «1С-Битрикс» для всего сайта.
