# MySQL

Конфигурация и init-скрипты для контейнера `mysql`.

| Параметр | Значение |
|----------|----------|
| Host (из хоста) | `localhost:3306` |
| Host (из backend) | `mysql:3306` |
| Database | `suzuki` |
| User / Password | `suzuki` / `suzuki` |
| Root password | `root` |

Init-скрипты: `init/*.sql` — выполняются при первом запуске контейнера.
